# Portfolio

A modern, animated developer portfolio built with React, Framer Motion, and Tailwind CSS.
Auto-populates from your GitHub profile and repositories.

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env` and add your GitHub token:
   ```
   VITE_GITHUB_TOKEN=your_token_here
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run locally:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- **React** + **Vite**
- **Framer Motion** — page & scroll animations
- **Tailwind CSS** — utility-first styling
- **Lucide React** — icon library
- **GitHub API** — dynamic data

## Deploy

Works great on Vercel, Netlify, or GitHub Pages.
